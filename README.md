### Usage

Windows users can use the exe.

1. Clone the repo
   ```sh
   git clone https://github.com/ReallyFatYoshi/mcbe-functions-transpiler.git
   ```
2. Install Pip packages
   ```sh
   pip install -r requirements.txt
   ```
3. Run the `source.py` file.
   ```sh
   python source.py
   ```
